# Disconnect-GitHubAccount

## SYNOPSIS
Disconnects from GitHub and removes the current GitHub configuration.

## SYNTAX

```powershell
Disconnect-GitHubAccount [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Disconnects from GitHub and removes the current GitHub configuration.

## EXAMPLES

### EXAMPLE 1

